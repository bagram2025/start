# 📓 Terminal Diary

Терминальный дневник с хранением записей в PostgreSQL. Пишите заметки прямо из консоли — с компьютера или смартфона.

## 🚀 Возможности

- ✍️ **Писать заметки** — многострочный ввод, сохраняется по пустой строке
- 📖 **Читать заметки** — вывод последних 20 записей с датами
- 📱 **Доступ со смартфона** — через SSH-клиент (Termux, iSH)
- 🗄️ **Хранение в PostgreSQL** — все записи в базе данных
- 🐧 **Работает на любом VPS** — лёгкий, без веб-сервера

## 🛠️ Стек

- **Python 3**
- **PostgreSQL**
- **psycopg2**
- **Bash** (для серверной части)

## 📋 Требования

- VPS с Ubuntu/Debian
- Python 3.8+
- PostgreSQL 12+
- Настроенный SSH-доступ
